# AudioGrabber

Convert videos to MP3 in one click.

## How to Use

1. **Download** — go to the [Releases](../../releases/latest) page and click `AudioGrabber.exe` to download.
2. **Double-click** `AudioGrabber.exe` to start the app (no installation needed).
3. **Select videos** — click **Pick a file** or **Pick a folder** (or drag videos onto the window) — the MP3s are made automatically.
4. **Wait** — the MP3s appear in a new "MP3" folder next to your videos.

That's it! Your original videos remain untouched.
